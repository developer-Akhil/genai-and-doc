# LangChain
LangChain is an open-source framework for building applications powered by Large Language Models (LLMs). It provides abstractions and orchestration tools to connect LLMs with external data sources, tools, memory, and workflows.

LangChain is an open-source framework designed to simplify building applications powered by large language models (LLMs). It provides modular tools to chain together components like prompts, memory, and data retrieval for more reliable AI workflows.
